# YouTube Timestamp

A TypeScript-based application for working with YouTube video timestamps, built with [Bun](https://bun.sh).

## Overview

This project provides tools for managing and working with YouTube video timestamps, enabling users to bookmark, track, and navigate specific moments in videos efficiently.

## Prerequisites

Before you begin, ensure you have the following installed:

- [Bun](https://bun.sh) v1.2.18 or higher

To install Bun:
```bash
curl -fsSL https://bun.sh/install | bash
```

## Installation

1. Clone the repository:
```bash
git clone https://github.com/enkee0/Youtube-Timestamp.git
cd Youtube-Timestamp
```

2. Install dependencies:
```bash
bun install
```

### Scripts

Run the main application:
```bash
bun run dev:api
bun run dev:web
```
